# Arquitetura de Computadores 👩🏽‍💻

### DESCRIÇÃO

Repositório com avaliações e listas de exercícios da disciplina de Arquitetura de Computadores, desenvolvidas em Assembly (MIPS). Os conteúdos estão organizados conforme os tópicos abordados em sala de aula.

### OBJETIVO

Este repositório tem como objetivo exercitar os conteúdos abordados em sala de aula, contribuindo para o desenvolvimento das minhas habilidades ao longo do curso.

### ESTRUTURA

l1-entrada-saida-operadores: Exercícios de entrada, saída e operações matemáticas, com foco na prática da sintaxe do Assembly (MIPS) e no desenvolvimento do raciocínio lógico.

### TECNOLOGIAS UTILIZADAS

- Assembly (MIPS)
- MARS
- JDK
- Git e GitHub

### COMO EXECUTAR

1. Certifique-se de que o JDK está instalado
2. Certifique-se de que o MARS está instalado
3. Abra o arquivo .asm no MARS
4. Compile o código: pressione F3 (Assemble)
5. Execute o programa: pressione F5 (Run → Go)

Os exercícios fazem parte do meu aprendizado acadêmico e, como todo universitário, ainda estou em processo de aprendizagem. Isso significa que os códigos podem ser atualizados conforme meus conhecimentos evoluem. 😊
